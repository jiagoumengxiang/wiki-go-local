# Roadmap: Wiki-Go 隐藏文件读取功能

**Created:** 2025-04-05

## Overview

This roadmap breaks down the hidden files feature into 4 phases, each building on the previous one. All phases have been completed and shipped in v1.0.

## Milestones

### ✅ v1.0 隐藏文件读取功能 — SHIPPED 2026-04-05

All 4 phases completed successfully. See [milestones/v1.0-ROADMAP.md](milestones/v1.0-ROADMAP.md) for full details.

**What was shipped:**
- File system access to hidden folders (.config, .system, etc.)
- Navigation, editing, attachments, and versions support for hidden folders
- Search and permission system verified for hidden folders
- 18/18 requirements completed

**Phase breakdown:**
- Phase 1: 文件系统基础修改 — File system modifications
- Phase 2: 用户界面和文档操作 — UI and document operations
- Phase 3: 文件附件和版本历史 — Attachments and version history
- Phase 4: 搜索和权限控制 — Search and permissions
